## Project Description

Student Management System is a Spring Boot application used to manage student records and marks. The system provides CRUD operations for student details and stores student information in a MySQL database.

## Technologies Used

* Java
* Spring Boot
* Spring Data JPA
* MySQL
* Maven

## Features

* Add Student
* View Students
* Update Student Details
* Delete Student Details
* Store Student Marks

## API Endpoints

### Add Student

POST /students

### View Students

GET /students

### Update Student

PUT /students/{id}

### Delete Student

DELETE /students/{id}

## Conclusion

This project demonstrates the implementation of a Student Management System using Spring Boot and MySQL with CRUD functionality for managing student records efficiently.
